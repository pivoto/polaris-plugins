# polaris-toolkit

- polaris-core

  常用工具库，只存在Slf4j日志、JSR标准依赖，无更多的三方依赖，保待纯净

- polaris-extra-*

  扩展工具库，针对特定用途或功能的扩展工具，有具体的功能依赖

- polaris-builder

  代码生成器工具
